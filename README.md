Advanced Filament Estimation Utility
========================================================================
Version: 1.0.0

Release Date: January 2026

------------------------------------------------------------------------
DESCRIPTION
------------------------------------------------------------------------
Ungoliant is a high-precision desktop utility designed for 3D printing 
professionals and enthusiasts. It provides real-time estimation of 
remaining filament based on weight or visual percentage, helping you
avoid failed prints due to running out of material.

------------------------------------------------------------------------
KEY FEATURES
------------------------------------------------------------------------
- SPOOL ANALYSIS: Dual-mode input for precision weighing or manual 
  percentage estimation.
- REAL-TIME VISUALIZER: Dynamic spool display that drains as you input
  print requirements.
- COST TRACKING: Live calculation of print costs based on material weight.
- MATERIAL LIBRARY: Optimized presets for PLA, PETG, ABS, and more.
- LOW FILAMENT WARNING: Visual alerts when your spool cannot sustain
  the required print.

------------------------------------------------------------------------
USAGE INSTRUCTIONS
------------------------------------------------------------------------
1. Select your material type to set the correct density.
2. Enter your "Gross Weight" (Spool + Filament) or toggle 
   "Estimate by %" if you don't have a scale.
3. Input the required weight from your slicer software.
4. Review the "Remaining After Print" and "Estimated Cost" results.
5. Watch the spool visualizer to see the predicted fill level.
